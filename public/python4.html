03-01. 변수 만들기

프로그래밍 언어의 핵심은 값을 기억하고 연산할 수 있다는 점입니다. 파이썬에서는 변수에 값을 할당하여 다양한 연산을 수행할 수 있습니다.

기본 변수 할당

x = 77
print(x + 3)

결과: 80

= 는 '같다'가 아니라 '오른쪽 값을 왼쪽 변수에 할당한다'는 의미입니다.

문자열 할당

x = "Hello World"
print(x)

결과: Hello World

변수에는 숫자 뿐 아니라 문자열도 저장할 수 있습니다.

변수 재할당

x = "Hello World"
print(x)

x = 10000
print(x)

결과:
Hello World
10000

같은 변수 이름에 다른 값을 할당하면, 이전 값은 덮어씌워집니다.

변수 사용 전 오류

print(x)

x = 50

NameError: name 'x' is not defined

변수를 사용하려면 반드시 먼저 값을 할당해야 합니다.

여러 변수 한 번에 할당

a, b, c = "Hello", "World", "!!!"
print(a)
print(b)
print(c)

결과:
Hello
World
!!!

,(쉼표)를 이용하여 여러 변수에 값을 동시에 할당할 수 있습니다.

변수 값 바꾸기 (Swap)

a, b, c = "Hello", "World", "!!!"
b, c, a = a, b, c
print(a)
print(b)
print(c)

결과:
!!!
Hello
World

파이썬은 간단하게 여러 변수의 값을 서로 바꾸는 것도 가능합니다.

변수 작명 규칙

  • 숫자, 영어, 한글 등 사용 가능 (UTF-8 문자)
  • 특수문자 사용 불가 (+, -, *, / 등)
  • 대소문자 구분 (num과 Num은 다른 변수)
  • 숫자로 시작 불가 (ex. 1name ❌, name1 ✅)
  • 파이썬 예약어는 변수로 사용 불가 (if, for, while 등)
