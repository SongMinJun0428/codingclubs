파이썬 기초 - 변수 예제

예제 1: 변수 값 바꾸기

문제: a → c, b → a, c → b로 바꾼 뒤 출력

a, b, c = 27, 0, -3
a, b, c = c, a, b
print(a, b, c)

0 -3 27

예제 2: 변수 타입 확인

a, b, c, d = 0, '', 6 / 3, None
print(type(a), type(b), type(c), type(d))

<class 'int'> <class 'str'> <class 'float'> <class 'NoneType'>

예제 3: 문자열 합치기 후 변수 삭제

a, b = 'incoding', 'plus'
c = a + b
del a, b
print(c)

incodingplus

예제 4: 누적 연산자 사용

a, b, c = 0, 1, 5
a += 1
b += 2
c -= 2
print(a, b, c)
a += 1
b *= 3
c -= 2
print(a, b, c)
a += 1
b *= 3
c -= 2
print(a, b, c)

1 3 3
2 9 1
3 27 -1

예제 5: sep 사용

a, b, c = 36, 200, -24
print(a, b, c, sep=", ")

36, 200, -24

예제 6: end 사용

a, b, c = 36, 200, -24
print(a, end=", ")
print(b, end=", ")
print(c)

36, 200, -24

예제 7: 특수문자 출력

print('\'\"')

'"

예제 8: + 연산으로 문자열 출력

a = '2024년'
b = '갑진년'
print(a + '은 ' + b + '입니다.')

2024년은 갑진년입니다.

예제 9: f-string으로 2줄 출력

name = "홍길동"
age = 99
print(f"저의 이름은 {name}이고,")
print(f"나이는 {age}세입니다.")

저의 이름은 홍길동이고,
나이는 99세입니다.
