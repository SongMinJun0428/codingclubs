bool 타입이란?

불(bool)은 참(True) 또는 거짓(False)을 나타내는 값으로, 조건 판단에서 자주 사용됩니다.

예제: bool 값 출력

a = True
print(a)
b = False
print(b)

True
False

비교 연산자 종류

연산자의미
a == b같을 때 True
a != b다를 때 True
a > b클 때 True
a < b작을 때 True
a >= b크거나 같을 때 True
a <= b작거나 같을 때 True

주의: =과 ==는 전혀 다른 의미입니다.

숫자 비교

print(1 == 1)
print(1 == 2)

a = 1
b = 2

print(a != b)
print(a != (b - 1))
print(a > b)
print(a < b)
print(a >= b)
print(a >= (b - 1))

True
False
True
False
False
True
False
True

문자 비교

a = 'python'
b = 'python'
c = 'dython'
d = 'Python'

print(a == b)
print(a == c)
print(a != d)

True
False
True

문자 비교 (크기 비교)

a = 'python'
b = 'Python'
c = 'ppP'
d = 'pPp'

print(a > b)
print(c <= d)
print(a >= c)

True
False
True

ord() 함수로 문자 크기 확인

print(ord('p'))
print(ord('P'))

112
80

문자 비교는 내부적으로 ASCII(유니코드) 값으로 비교됩니다.

문자열 비교는 앞글자부터

print('p' > 'Python')

True
