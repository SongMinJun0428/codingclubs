파이썬 기초 - is, in, type

is 연산자

is는 두 객체가 실제로 동일한 객체인지(같은 주소인지) 확인할 때 사용합니다.

id()를 이용한 객체 확인

print(id(1))
print(id(1))
print(id(1.0))

※ id는 실행 시마다 달라질 수 있음
같은 값은 같은 id를 갖고, 다른 객체는 다른 id를 가짐

is 와 == 비교

print(1 == 1.0)      # 값 비교
print(1 is 1.0)      # 객체 비교
print(1 is not 1.0)  # 객체가 다르면 True

True
False
True

in 연산자

in은 어떤 값이 시퀀스 안에 포함되어 있는지 확인할 때 사용합니다.

문자 포함 여부 확인

print('a' in 'apple')
print('s' in 'apple')
print('s' not in 'apple')

True
False
True

문자열도 in으로 검사 가능

print('ap' in 'apple')
print('al' in 'apple')

True
False

type 비교

type() 함수는 값의 자료형을 알려줍니다. 이 값은 is나 ==로 비교할 수 있습니다.

type()과 is 비교

a = 10
b = 20
c = 1.3

print(type(a))            # 
print(type(a) is int)     # True
print(type(a) is type(b)) # True
print(type(a) is type(c)) # False

<class 'int'>
True
True
False

type()과 == 비교

print(type(a) == int)
print(type(a) == type(b))
print(type(a) == type(c))

True
True
False

type()의 결과는 하나의 고정된 객체이므로 ==도 동일하게 작동합니다.
