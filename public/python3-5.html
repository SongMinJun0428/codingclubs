예제 4: 3의 배수이면 1, 아니면 0

num = int(input())
print(int(num % 3 == 0))

입력: 6 → 1
입력: 0 → 1

예제 5: 2 또는 3의 배수이면 1, 아니면 0

num = int(input())
print(int(num % 2 == 0 or num % 3 == 0))

입력: 8 → 1
입력: 7 → 0

예제 6: 단락 평가 예측

print((1 / 2) and (3 - 4))       # 0.5 → True → and → -1
print('' or None)                # '' → False → or → None
print((1 // 2) and (3 / 5))      # 0 → False → and → 0
print('_' or (3 // 5))           # '_' → True → or → '_'

-1
None
0
_
