사칙연산

파이썬에서는 숫자의 계산을 할 수 있습니다. 숫자는 정수(int)와 실수(float)로 나눠집니다. 사칙연산은 다음과 같은 기호들을 사용합니다.

연산자 설명
a + b a 변수의 값과 b 변수의 값을 더한 결과가 나온다.
a - b a 변수의 값에서 b 변수의 값을 뺀 결과가 나온다.
a * b a 변수의 값과 b 변수의 값을 곱한 결과가 나온다.
a / b a 변수의 값에서 b 변수의 값을 나눈 결과가 나온다. 결과는 실수(float)로 나온다.
a // b a 변수의 값에서 b 변수의 값을 나눈 몫이 나온다.
a % b a 변수의 값에서 b 변수의 값을 나눈 나머지가 나온다.
a ** b a 변수를 b번 곱한 결과가 나온다. (제곱)

예제 1: 덧셈

print(30 + 10)

결과: 40

예제 2: 뺄셈

print(30 - 10)

결과: 20

예제 3: 곱셈과 나눗셈

print(30 * 10)
print(10 / 5)

결과:
300
2.0

예제 4: 몫과 나머지

print(10 // 5)
print(5 // 3)
print(5 % 3)

결과:
2
1
2

예제 5: 제곱

print(10 ** 3)
print(2 ** 10)
print(2 ** 16)

결과:
1000
1024
65536
