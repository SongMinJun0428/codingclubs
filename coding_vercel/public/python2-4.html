03-04. 값 입력 받기

사용자로부터 값을 입력받는 것은 파이썬 프로그래밍에서 매우 중요한 기능입니다. input() 함수를 통해 입력을 받을 수 있습니다.

기본 입력

input()

프로그램이 일시 정지되며 입력을 기다립니다.

입력 후 출력

print(input())

Hello → Hello

입력한 값이 그대로 출력됩니다.

input과 괄호

print(input)

<built-in function input>

()를 생략하면 함수가 호출되지 않고 함수 그 자체를 출력합니다.

변수에 입력 저장

a = input()
print(a)

입력값이 변수 a에 저장되어 출력됩니다.

입력값과 숫자 연산 오류

a = input()
print(a + 30)

TypeError: can only concatenate str (not "int") to str

입력값은 항상 문자열(str)로 저장되므로 숫자 연산 시 오류가 발생합니다.

입력값을 숫자로 변환

a = input()
print(int(a) + 30)

입력값을 정수형으로 변환하면 연산이 가능합니다.

여러 값 입력받기

a = int(input())
b = int(input())
print(a + b)

입력값이 정수로 변환되어 연산 결과가 출력됩니다.

입력 시 설명 추가하기

a = input('물량을 입력해주세요 : ')
print(a)

물량을 입력해주세요 : 30
30

input() 안에 문자열을 넣으면 입력 전에 설명 문구를 출력할 수 있습니다.
