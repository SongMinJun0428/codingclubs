03-02. 변수 다루기

변수에 값을 할당하는 것 외에도 파이썬에서는 변수의 타입을 확인하거나 삭제하거나, 초기화 없이 선언하는 방법도 제공합니다.

type 함수로 변수 타입 확인하기

print(3 / 1)
print(type(3 / 1))

3.0
<class 'float'>

숫자끼리의 연산도 결과에 따라 float 타입이 될 수 있습니다. type() 함수를 사용하여 타입을 확인할 수 있습니다.

예제: 나눗셈 결과 비교

x, y = 30, 40
print(x / y)
print(type(x / y))

print(x // y)
print(type(x // y))

0.75
<class 'float'>
0
<class 'int'>

/는 실수 나눗셈, //는 정수 나눗셈(몫)을 의미합니다.

변수 삭제하기 (del)

x = 50
print(x)

del x
print(x)

50
NameError: name 'x' is not defined

del 키워드를 사용하면 변수가 완전히 삭제됩니다. 이후에 다시 사용하면 오류가 발생합니다.

None 값 사용하기

x = None
print(x)

x = 30
print(x)

None
30

None은 "아직 값이 없음"을 표현할 때 사용합니다. 나중에 실제 값을 대입할 수 있습니다.
