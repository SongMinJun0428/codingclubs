실전 예제: 음수면 -1, 아니면 1 출력

입력한 숫자가 음수면 -1, 아니면 1을 출력하는 프로그램을 작성해봅시다.

기본 비교만 사용할 때

num = int(input())
x = int(num >= 0)
print(x)

입력값이 10이면 → 1
입력값이 -10이면 → 0

1 또는 -1 출력

num = int(input())
x = 2 * int(num >= 0) - 1
print(x)

입력값이 10이면 → 1
입력값이 -10이면 → -1

int(True) = 1 이므로 2*1 - 1 = 1, 2*0 - 1 = -1
