03-05. 다양한 출력 방법

print() 함수는 단순히 값을 출력하는 것 외에도 다양한 방식으로 출력을 조절할 수 있습니다.

여러 값 출력

print("Hello", "Incoding")

Hello Incoding

콤마(,)로 구분하여 여러 값을 출력할 수 있습니다. 기본적으로 값 사이에는 공백이 들어갑니다.

sep: 구분자 설정

print(1, 2, 3)
print(1, 2, 3, sep='H')
print(1, 2, 3, sep=", ")

1 2 3
1H2H3
1, 2, 3

sep은 값 사이에 들어갈 문자를 지정합니다.

end: 출력 후 끝나는 문자

print(1)
print(2)
print(3)

print(1, end=" ")
print(2, end=" ")
print(3, end=" ")

1
2
3
1 2 3

end는 기본적으로 줄바꿈(\n)이지만 다른 문자로 바꿀 수 있습니다.

줄바꿈 문자와 특수문자

print("A\nB\nC")
print("---")
print('A', 'B', 'C', sep='\n')
print('역슬래시 : \\')
print('작은 따옴표 : \'')
print("큰 따옴표 : \"")

A
B
C
---
A
B
C
역슬래시 : \
작은 따옴표 : '
큰 따옴표 : "

문자열 합치기

a = 'abc'
b = 'def'
print(a + b)

a = 'Hello'
b = 'Incoding'
print(a + ' ' + b)

abcdef
Hello Incoding

+ 연산자는 문자열을 이어붙이는 역할을 합니다.

f-string

a = 'Hello'
b = 'Incoding'
print(f"{a} {b}")

year = 1994
month = 8
day = 3
print(f"{year}년 {month}월 {day}일")

Hello Incoding
1994년 8월 3일

f-string은 문자열 안에 변수를 쉽게 삽입할 수 있습니다.

여러줄 문자열

onetwothree = """1
2
3"""
print(onetwothree)

name = "홍길동"
age = 30
print(f"""이름 :
  {name}님
나이 :
  {age}살""")

1
2
3
이름 :
  홍길동님
나이 :
  30살

큰 따옴표 3개(""")를 사용하면 여러 줄의 문자열을 표현할 수 있고, f-string과 함께 사용할 수 있습니다.
