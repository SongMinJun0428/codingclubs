논리 연산자란?

논리 연산자는 여러 개의 bool 값들을 조합하여 조건을 판단할 수 있게 해주는 연산자입니다.

and

print(True and True)
print(False and True)
print(True and False)
print(False and False)

True
False
False
False

and는 양쪽 모두가 True일 때만 결과가 True입니다.

or

print(True or True)
print(False or True)
print(True or False)
print(False or False)

True
True
True
False

or는 둘 중 하나라도 True이면 결과가 True입니다.

not

print(not True)
print(not False)

False
True

not은 bool 값을 반대로 바꿔줍니다.

논리 연산자 적용 예제

a = 5
print(10 > a or a > 20)
print(10 < a and a < 20)
print(not a > 5)

a = 6
print((a % 2 == 0 and a % 3 != 0) or (a % 12 == 0))

a = 12
print((a % 2 == 0 and a % 3 != 0) or (a % 12 == 0))

True
False
True
False
True

복잡한 조건들도 논리 연산자로 표현할 수 있습니다.

단락 평가 (Short-circuit Evaluation)

and와 or는 단순한 bool 계산 외에도 다른 타입의 값에서도 특정 규칙에 따라 동작합니다.

단락 평가 예제

print(1 and 3)            # 둘 다 True → 뒤의 값 반환
print(1 or 3)             # 앞이 True → 앞의 값 반환
print('---')
print('abc' and '')       # 앞이 True, 뒤가 False → '' 반환
print('abc' or '')        # 앞이 True → 'abc' 반환
print('---')
print('' or 3)            # 앞이 False → 3 반환
print('' and 1)           # 앞이 False → '' 반환
print('---')
print(0 and 123)          # 앞이 False → 0 반환
print(0 or -32)           # 앞이 False → -32 반환

3
1
---

abc
---
3

---
0
-32

단락 평가 규칙 정리

  • a and b: a가 False이면 a 반환, a가 True이면 b 반환
  • a or b: a가 True이면 a 반환, a가 False이면 b 반환
